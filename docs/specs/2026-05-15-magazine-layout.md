# 设计规范：杂志画报风与错落式网格布局

- **主题**：通过视觉权重区分资讯重要性，实现“高分内容”大屏重点展示。
- **日期**：2026-05-15
- **状态**：草案

## 1. 概述
本设计旨在打破整齐划一的网格排版，引入“权重展示”概念。评分高于 90 的高分资讯将自动升级为“英雄位（Hero Status）”，占据更大的空间和更醒目的视觉样式，形成类似专业科技杂志的排版韵律。

## 2. 核心特征

### 2.1 动态网格权重 (App.vue)
- **逻辑**：在渲染资讯流时，检测 `item.score`。
- **布局规则**：
    - `score >= 90`：在 PC 端占据 2 列宽度 (`col-span-2`)，展示超大预览图。
    - `score < 90`：占据 1 列宽度，保持现有的列表/卡片样式。
- **响应式**：在移动端，所有项自动回归为全宽单列显示。

### 2.2 英雄卡片样式 (NewsCard.vue)
- **视觉增强**：针对 col-span-2 的卡片，应用以下增强：
    - **背景流光**：边框或背景应用微弱的蓝色渐变流光。
    - **排版升级**：标题字号加大，摘要内容显示行数增多。
    - **沉浸媒体**：视频或大图将占据卡片的一半以上空间，形成强烈的视觉冲击。
- **交互**：悬停时，英雄卡片的阴影深度和缩放比例比普通卡片更为显著。

## 3. 技术实现

### 3.1 逻辑层
- 在 `App.vue` 的 `v-for` 循环中，根据 `item.score` 动态绑定 CSS 类：
  ```vue
  :class="item.score >= 90 ? 'lg:col-span-2' : 'col-span-1'"
  ```

### 3.2 表现层
- 为 `NewsCard.vue` 增加一个 `isFeatured` prop。
- 当 `isFeatured` 为真时，切换为另一种 Flex 布局或 CSS Grid 子布局。

## 4. 实施计划

1. **第一阶段：网格适配**
   - 调整 `App.vue` 的主网格容器，确保其具备多列跨度能力。
2. **第二阶段：英雄组件开发**
   - 在 `NewsCard.vue` 中实现大画幅模式的排版（左图右文或上图下文）。
3. **第三阶段：视觉细节润色**
   - 添加高分资讯特有的动态渐变边框和精致的背景阴影。

## 5. 验收标准
- [ ] 评分 90+ 的资讯在 PC 端明显比普通资讯大，且排版美观。
- [ ] 网格布局在资讯交错时不会出现大面积空白。
- [ ] 移动端显示依然紧凑顺滑。
