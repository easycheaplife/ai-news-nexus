# AI News Nexus Landing Page (Strategic Magazine Style) 设计文档

## 1. 目标与定位
将 AI News Nexus 打造为 **AI 时代的战略内参杂志**。它不仅仅是一个聚合器，更是一个能够从全网噪音中提炼出“核心信号”的权威智库。

## 2. 视觉规范 (Brand Identity)
采用 **“高对比度印刷美学”**，结合现代数字媒体的视觉张力。

- **核心配色**:
  - `Midnight` (背景): `#050505` (极致深黑)
  - `Action` (点缀): `#6366f1` (靛蓝色，用于高亮和关键动作)
  - `Ink` (主要文字): `#F5F5F5` (高级铜版纸白)
  - `Muted` (次要文字): `#94a3b8` (石板灰)
- **排版 (Typography)**:
  - **标题 (Display)**: 使用 **Playfair Display** (或类似的衬线体)，强调权威与叙事感。
  - **正文 (Body)**: 使用 **Inter**，确保在深色模式下的极佳可读性。
- **布局 (Layout)**:
  - **Grid-First**: 采用非对称的模块化网格，模仿顶级智库报告或《The Verge》的排版。

## 3. 页面模块 (The Intelligent Grid)

### 3.1 Hero Grid (首屏网格)
- **A 模块 (The Lead)**: 巨大的标题 "AI IS NOISY. NEXUS IS SIGNAL."，衬线体展示，背景是今日日报的模糊艺术化处理。
- **B 模块 (Resonance)**: 展示一个“共振频率”视觉组件，显示当前全网讨论热度最高的话题。
- **C 模块 (The Ticker)**: 底部横向滚动的实时信号流（Twitter, HF, Labs 等来源的抓取日志）。

### 3.2 Feature Stories (核心能力)
将功能包装为“深度报道”模块：
- **"The Art of Synthesis"**: 介绍如何将数千条信号聚类。
- **"Autonomous Intelligence"**: 介绍信源的自动发现与面试机制。
- **"Dual Perspective"**: 强调 Toxic 与 Official 双重叙事。

### 3.3 Tech Stack (工艺展示)
一个模拟“情报终端”的黑框，展示系统后端正如何处理全球情报，体现透明度与可控性。

## 4. 技术实现
- **字体引入**: Google Fonts (Playfair Display & Inter)。
- **布局引擎**: CSS Grid (用于实现复杂的非对称网格)。
- **动画**: 极简的淡入淡出和微弱的位移，保持“智库”的稳重感。

## 5. 验收标准
1. 页面展现出一种超越普通“生产力工具”的媒体感和权威感。
2. 衬线体标题在各种屏幕尺寸下均具有极强的视觉冲击力。
3. 网格布局在移动端能优雅地折叠为单列。
