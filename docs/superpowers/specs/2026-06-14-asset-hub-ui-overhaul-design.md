# 设计规范：资产库 UI 视觉重塑 (靛蓝统一风格)

**日期：** 2026-06-14  
**状态：** 草案  
**主题：** 将“方向 2”(百科与白皮书) 的视觉风格与“AI 情报中心”主仪表盘进行深度对齐。

## 1. 背景与目标
目前实现的“技术百科”和“深度白皮书”页面使用了较为通用的深色主题，缺乏主仪表盘那种精密的“情报中心”质感。本次重塑旨在统一全平台的视觉语言，采用项目核心的“靛蓝/青色”调色盘，并引入“结构化枢纽 (Structured Hub)”布局。

## 2. 视觉方向：“结构化枢纽”
- **主色调：** 靛蓝 (`#6366f1`) —— 用于核心高亮、边框和激活状态。
- **点缀色：** 青色 (`#22d3ee`) —— 用于数据标签、扫描线和共振指标。
- **背景色：** 深邃黑 (`#0a0a0c`) —— 与主仪表盘的深度感保持一致。
- **字体：** 
    - 展示标题：Inter (Bold/Black) —— 增强权威感。
    - 技术元数据：JetBrains Mono —— 用于技术规格和编号。
- **形状：** 大圆角 (`rounded-[2.5rem]`) —— 保持“流体现代”的整体感。

## 3. 组件详细设计

### 3.1 AI 技术百科 (`Wiki.vue`)
- **布局：** 响应式三栏网格。
- **卡片样式 (`term-card`)：**
    - 半透明深色背景，带有 1px 靛蓝调色边框。
    - 右上角加入“呼吸灯 (Glow Dot)”动效，指示资产的实时活跃度。
    - 悬停效果：卡片轻微上浮，边框亮度提升，伴有淡紫色阴影。
- **头部：** 采用靛蓝垂直条装饰，中英文双语标题对齐。

### 3.2 深度战略白皮书 (`Whitepapers.vue`)
- **布局：** 单栏垂直堆叠（“卷宗”风格）。
- **列表项样式 (`dossier-card`)：**
    - 左侧 4px 厚边的靛蓝装饰条，悬停时加粗。
    - 右上角加入“文件折角”视觉处理。
    - 清晰的元数据分区（统计周期、合成状态、资产数量），使用等宽字体增强专业感。
- **阅读体验：** 
    - 全屏沉浸式弹窗，优化排版（基于 Tailwind Typography/Prose）。
    - 针对战略要点进行高亮处理，提升长文阅读的重点捕捉效率。

## 4. 技术实现
- **框架：** Vue 3 (SFC)。
- **样式：** Tailwind CSS (Utility-first)，结合 scoped CSS 实现“扫描线”和“呼吸灯”等自定义动效。
- **图标：** Lucide-vue-next (沿用现有库)。
- **数据流：** 保持现有的 Axios 调用逻辑，请求 `/api/assets/terms` 和 `/api/assets/reports`。

## 5. 验收标准
1. 首页仪表盘与资产库页面之间无视觉断层。
2. 白皮书的呈现更具“权威感”和“易读性”。
3. 搜索和筛选功能在全新样式下依然表现流畅。
